Publishing Tables and Figures from didhelpers

Overview

This vignette demonstrates how to turn didhelpers diagnostic output into publication-quality tables and figures suitable for academic papers, slide decks, and reports.

We cover four output paths:

  1. gt / kableExtra tables via as_table()
  2. broom-style tidy output via tidy(), glance(), augment()
  3. Regression tables via modelsummary (LaTeX, Word, HTML)
  4. Customized ggplot2 figures for publication

Setup

library(didhelpers)
library(ggplot2)
data("staggered_data")

# Run diagnostics
pt <- test_pretrends(staggered_data, unit, time, outcome, treat, n_lags = 3)
#> Wald test, H0: joint nullity of lead_3 and lead_2
#>  stat = 0.232853, p-value = 0.792378, on 2 and 400 DoF, VCOV: IID.
bs <- bacon_weight_summary(staggered_data, unit, time, outcome, treat)

To export directly to a Word document:

# Save to .docx (requires flextable)
export_pretrends_table(pt, output = "my_table.docx")

You can also call modelsummary directly for more control:

library(modelsummary)
modelsummary(
  list("Event-Study" = pt),
  stars = c("*" = 0.1, "**" = 0.05, "***" = 0.01)
)

modelsummary is the best fit because it uses tidy() and glance() automatically, supports all output formats, and integrates with gt and kableExtra.

4. Publication-Ready Figures

All plot methods return ggplot2 objects that can be further customized with standard ggplot2 layers.

Event-study plot

autoplot(pt) +
  theme_classic(base_size = 12) +
  labs(title = NULL, caption = NULL) +
  theme(legend.position = "none")

Adjusting confidence level

autoplot(pt, conf.level = 0.90) +
  theme_classic(base_size = 12) +
  labs(title = "Event-Study (90% CI)")

Bacon decomposition plot

autoplot(bs) +
  theme_classic(base_size = 12) +
  labs(title = NULL) +
  theme(legend.position = "bottom")

Composite report figure

report <- did_diagnostics_report(staggered_data, unit, time, outcome, treat)
#> Wald test, H0: joint nullity of lead_3 and lead_2
#>  stat = 0.232853, p-value = 0.792378, on 2 and 400 DoF, VCOV: IID.
plot(report)
